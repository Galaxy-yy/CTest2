#Week4 Tasks

##1.第k个数

描述
```
输入正整数n和k，输出n中从右端开始的第k个数字的值（k从1开始）。将求n中右端第k个数字定义成函数，
如果超过了n的位数，则函数返回-1，否则返回n中第k个数字。
本组包含多组测试数据。
输入格式说明：每组测试数据输入占一行，包括两个正整数n,k，n在1-4000000000闭区间内，
遇文件尾测试结束。
输出格式说明：每组测试样例的输出结果占一行。
```

样例输入
```
321 3
421 4
42 12
```

样例输出
```
3
-1
-1
```

##2.冰雹数

描述
```
n(0)是一个给定的正整数，对于i=0,1,2,...定义(1)若n(i)时候偶数，则n(i+1)/2；
(2)若n(i)是奇数，则n(i+1)=3n(i)+1；(3)若n(i)是1，则序列结束。
本组包含多组测试数据。
输入格式说明：每组测试数据输入占一行，包括一个正整数n0，n0在1-4000000000闭区间内，
遇文件尾测试结束。
输出格式说明：输出冰雹数序列以及序列数的总个数，相邻两组测试样例的输出结果空一行。
```

样例输出
```
77
92
```

样例输出
```
Hailstone generated by 77:
77 232 116 58 29 88 44 22 11 34 17 52 26 13 40 20 10 5 16 8 4 2 1
Number of hailstone generated:23

Hailstone generated by 92:
92 46 23 70 35 106 53 160 80 40 20 10 5 16 8 4 2 1
Number of hailstone generated:18
```

##3.掷骰子游戏

描述
```
编写一个模拟“掷骰子游戏”的程序。游戏规则：每轮投两次，取两次的和，第一轮若和为7或11则获胜，
游戏结束；若和为2，3，或12则输，失败结束；若和为其他，则将此值作为自己的点数，继续第二轮...
直到某轮的和等于该点数则获胜，若出现7，则输掉游戏。模拟每投一次骰子的随机取数规则：输入一个
非负整数，称为启动数，你需要计算得到启动数各位数之和记为sum，现将1-6这张数字卡片按顺时针方向
摆放一个环，一只蚂蚁每次从数字为1的卡片摆放位置出发，这时你告诉蚂蚁按顺时针方向走sum步，蚂蚁
最终停留位置上的卡片数字即视为本次投骰子得到的点数。
本组包含多组测试数据。
输入格式说明：第一行输入n表示测试样例的个数，后面紧接着有n行输入，每行输入包含两个非负整数
记为a，b，整数范围在0-10000闭区间之内，视为第一轮两次投掷的启动数，若进行到了第n轮时，则第n轮
两次投掷的启动数分别看作是a+n-1，b+n-1。
输出格式说明：每组测试样例的输出结果占一行，输出本次游戏的最终结果。
```

样例输入
```
3
1234 1
1 42
0 422
```

样例输出
```
success！
fail！
success！
```

##4.阶乘倒数累加

描述
```
编写程序，计算s=1+(1/2!)+(1+3!)+...+(1/n!)。将计算n！定义成函数。
输入：有多组输入，每组输入对应一行，每行为正整数n。当输入的n为0时程序结束运行。
输出：每行输入对应一行输出，保留8位小数，当输入不合法时输出Error！
```

样例输入
```
5
1
-2
10
15
0
```

样例输出
```
s = 1.71666667
s = 1.00000000
Error！
s = 1.71828180
s = 1.71828183
```
